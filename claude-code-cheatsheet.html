🖥️ CLI 与标志
核心命令
claude 交互式
claude "q" 带提示
claude -p "q" 无头模式
claude -c 继续上次
claude -r "n" 恢复
claude update 更新
重要标志
--model 设置模型
-w Git worktree
-n / --name 会话名称
--add-dir 添加目录
--agent 使用代理
--allowedTools 预批准
--output-format json/stream
--json-schema 结构化
--max-turns 限制轮次
--max-budget-usd 费用上限
--console 通过 Anthropic Console 认证
--verbose 详细模式
--bare 最小化无头（无 hooks/LSP）NEW
--channels 权限中继 / MCP 推送NEW
--remote Web 会话
--chrome Chrome
权限模式： default 提示确认 · acceptEdits 自动接受编辑 · plan 只读 · dontAsk 未允许则拒绝 · bypassPermissions 跳过所有
重要环境变量： ANTHROPIC_API_KEY · ANTHROPIC_MODEL · CLAUDE_CODE_EFFORT_LEVEL (low/med/high) · MAX_THINKING_TOKENS (0=关闭) · CLAUDE_CODE_MAX_OUTPUT_TOKENS (默认 32K) · CLAUDE_CODE_DISABLE_CRON
