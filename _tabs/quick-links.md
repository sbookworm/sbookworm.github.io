### 快捷键速查

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+C` | 取消输入/生成 |
| `Ctrl+D` | 退出会话 |
| `Ctrl+L` | 清屏 |
| `Ctrl+O` | 切换详细输出 |
| `Shift+Tab` | 循环切换权限模式 |
| `Alt+T` | 切换思考模式 |
| `Alt+P` | 切换模型 |

### 常用命令

- `/clear` - 清除对话
- `/compact` - 压缩上下文
- `/model` - 切换模型
- `/memory` - 编辑 CLAUDE.md
- `/mcp` - 管理 MCP 服务器
