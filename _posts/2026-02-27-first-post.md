## 关于这个博客

我用 GitHub Pages + Jekyll 搭建了这个博客。

### 为什么会用 GitHub Pages？

1. **免费托管** - 不需要服务器费用
2. **支持自定义域名** - 可以用自己的域名
3. **Markdown 支持** - 用熟悉的语法写文章

### 怎么写文章？

在 `_posts` 目录下创建 Markdown 文件，文件名格式：

```
YYYY-MM-DD-标题.md
```

文件开头需要添加 front matter：

```yaml
---
layout: post
title: 你的标题
date: 2026-02-27
tags: 标签
---

正文内容...
```
